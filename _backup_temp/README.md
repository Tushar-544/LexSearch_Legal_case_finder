## рџљЂ Quick Start

### Step 1 вЂ” Prepare data (Google Colab)

Upload the project to Colab and run notebooks **in order**:

```
01_data_cleaning.ipynb    в†’ data/processed/cases_clean.csv
02_document_builder.ipynb в†’ data/processed/documents.jsonl
03_chunking.ipynb         в†’ data/processed/chunks.jsonl
04_embeddings.ipynb       в†’ vector_db/faiss_index/
```

Download `vector_db/` and `data/processed/` back to your local machine.

---

### Step 2 вЂ” (Optional) Download + summarize PDFs

```bash
pip install -r requirements.txt
python src/pdf_processor.py --limit 1000
```

This will create `data/processed/pdf_summaries.jsonl`.

---

### Step 3 вЂ” Install Node dependencies

```bash
npm install
```

---

### Step 4 вЂ” Run the backend

```bash
node server/index.js
```

Server starts at **http://localhost:3001**

---

### Step 5 вЂ” Run the frontend

```bash
npm run dev:client
```

Frontend starts at **http://localhost:5173**

---

### Step 6 вЂ” Or run both at once

```bash
npm run dev
```

## рџ›  Tech Stack

| Layer        | Technology                                    |
|--------------|-----------------------------------------------|
| Embeddings   | `sentence-transformers/all-MiniLM-L6-v2`     |
| Vector DB    | FAISS (IndexFlatIP, cosine similarity)        |
| LLM          | `google/flan-t5-large` (free, local)          |
| Summarizer   | `facebook/bart-large-cnn` (free, local)       |
| PDF parsing  | PyMuPDF / pdfplumber                          |
| Backend      | Node.js + Express                             |
| Frontend     | React + TypeScript + Vite                     |

## вљ  Notes

- All PDF URLs are used **as-is** from the dataset вЂ” no prefix is added.
- Broken/inaccessible PDF links are skipped silently.
- No OpenAI API key required вЂ” all models are free and run locally.
- GPU is optional вЂ” everything runs on CPU (slower but functional).
